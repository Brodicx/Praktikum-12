Jurnal Investigasi Time Series

Antarmuka Streamlit untuk eksplorasi time series data.csv.

Cara Menjalankan Lokal

  1. Pastikan Python 3.9+ terpasang.
  2. Instal dependensi: 
    pip install streamlit pandas numpy seaborn statsmodels matplotlib
  3. Jalankan aplikasi: 
    streamlit run app.py
  4. Buka URL yang muncul (biasanya http://localhost:8501).

Fitur

  • Load & impute data time series, cek gap tanggal.
  • Plot data harian vs rata-rata mingguan.
  • Seasonal decomposition (observed, trend, seasonal, residual).
  • Rolling mean/std dan uji ADF untuk stasioneritas.
